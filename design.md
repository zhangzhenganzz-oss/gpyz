# 巴菲特-芒格价值投资分析软件设计文档

## 一、核心理论基础

### 1. 巴菲特投资原则
- **能力圈原则**: 只投资自己能理解的企业
- **护城河理论**: 寻找具有持久竞争优势的企业
- **安全边际**: 以低于内在价值的价格买入
- **长期持有**: 买入优秀公司并长期持有
- **复利效应**: 时间是优秀企业的朋友

### 2. 芒格投资原则
- **多元思维模型**: 运用多学科知识分析企业
- **逆向思维**: 先想如何失败，再避免失败
- **心理误判检查清单**: 避免认知偏差
- **优质企业溢价**: 为优秀企业支付合理价格

## 三、详细功能设计

### 1. 企业质量评估 (Quality Score)

#### 评估维度
- **盈利能力** (25%)
  - ROE > 15% (连续5年)
  - ROA > 10%
  - 毛利率 > 40%
  - 净利率 > 15%

- **盈利稳定性** (20%)
  - 净利润增长率标准差
  - 连续盈利年数
  - 经济周期中的表现

- **资本效率** (20%)
  - 资本回报率 (ROIC)
  - 投入资本回报率
  - 自由现金流/净利润

- **成长性** (15%)
  - 收入复合增长率 (5年)
  - 利润复合增长率 (5年)
  - 自由现金流增长

- **财务保守性** (20%)
  - 负债率 < 50%
  - 流动比率 > 1.5
  - 利息覆盖倍数 > 5

### 2. 护城河分析 (Moat Analysis)

#### 护城河类型识别
- **品牌护城河**: 品牌溢价能力、消费者心智份额
- **成本优势**: 规模经济、流程优化、资源优势
- **网络效应**: 用户增长带来的价值增长
- **转换成本**: 客户更换供应商的成本
- **监管壁垒**: 牌照、专利、特许经营权

#### 护城河强度评分
- 无护城河: 0-20分
- 窄护城河: 21-50分
- 宽护城河: 51-80分
- 极宽护城河: 81-100分

### 3. 内在价值计算

#### DCF模型 (现金流折现)
```
内在价值 = Σ (FCF_t / (1+r)^t) + 终值 / (1+r)^n

其中:
- FCF_t: 第t年的自由现金流
- r: 折现率 (通常8-12%)
- n: 预测期 (通常10年)
- 终值 = FCF_n × (1+g) / (r-g)
```

#### 简化估值方法
- **市盈率法**: 合理PE × 每股收益
- **市净率法**: 合理PB × 每股净资产
- **格雷厄姆公式**: V = EPS × (8.5 + 2g)

### 4. 安全边际计算

```
安全边际 = (内在价值 - 当前价格) / 内在价值 × 100%

买入建议:
- 安全边际 > 50%: 强烈买入
- 安全边际 30-50%: 买入
- 安全边际 10-30%: 观望
- 安全边际 < 10%: 高估
```

### 5. 管理层评估

#### 评估指标
- **资本配置能力**: ROE、再投资回报率
- **股东回报**: 分红政策、回购记录
- **透明度**: 财报质量、信息披露
- **诚信记录**: 历史承诺兑现情况
- **薪酬合理性**: 管理层薪酬与业绩挂钩

## 五、技术实现方案

### 前端技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- Chart.js 数据可视化
- LocalStorage 数据持久化

## 六、使用流程

1. **输入股票代码** → 系统自动获取财务数据
2. **查看质量评分** → 了解企业基本面质量
3. **分析护城河** → 判断竞争优势持久性
4. **计算内在价值** → DCF模型自动估值
5. **检查安全边际** → 确定买入时机
6. **心理偏见检查** → 避免投资决策错误
7. **加入投资组合** → 跟踪和管理持仓
